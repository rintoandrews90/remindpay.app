# HANDOFF — RemindPayWeb

_Last updated: 2026-07-06_

## Project

Static marketing site for **RemindPay** (iOS bill & warranty tracker app), live at https://remindpay.app.

- Single-page site: `index.html` (Tailwind via CDN, Inter font, no build step)
- `PrivacyPolicy.html`, `manifest.json`, `robots.txt`, `sitemap.xml`, `images/`
- Deployed to Cloudflare via Wrangler (`wrangler.jsonc`); custom domain via `CNAME`
- App Store: https://apps.apple.com/us/app/remindpay-bills-warranty/id1609633170

## Current state

- **`index.html`** uses the original blue design (primary `#2E6FA8`, Inter 800–900 display, pill buttons, gradient CTAs). A Claude-inspired cream/coral restyle was applied earlier in the session and then **reverted by the owner** — the blue design is the intended one.
- **`DESIGN.md`** documents the *actual* RemindPay design system (tokens for colors, typography, radii, shadows, components). It replaced an earlier Claude-branded file generated by `npx getdesign@latest add claude`.
- **`CLAUDE.md`** instructs agents to read `DESIGN.md` before any UI work. Keep this workflow: UI changes should follow DESIGN.md tokens.
